Punto de Venta - Vida / Accidentes Personales

Plataforma web que permite a los vendedores del canal realizar cotizaciones y emisiones de seguros de Vida y Accidentes Personales en tiempo real. Punto de Venta - Automóviles

Sistema especializado para la cotización, inspección y emisión de seguros de vehículos. El proceso inicia con la cotización donde se selecciona el paquete, datos del vehículo y coberturas adicionales. Posteriormente, se envía un link de auto-inspección al cliente para que capture fotos 360° del vehículo desde su celular. Una vez completada la inspección y creado el contacto, el vendedor procede a cargar documentos y emitir la póliza.

Proceso Automático - Archivo Único SFTP

El sistema verifica, valida según reglas de negocio y procesa según tipo. Maneja inconsistencias con código de colores: rojo (errores formato), amarillo (errores contacto), blanco (sin errores).

Conexiones por API

La respuesta incluye URLs de todos los documentos generados.
